# Sales

Generated by EIT Framework v2.5

## Stack
- Java 21
- Spring Boot 3.2.x
- Maven

## Run
```bash
./mvnw spring-boot:run
```

Server starts on **http://localhost:8080**

## Endpoints
- `GET  /api/health` — health check
- `POST /api/auth/login` — login

## Configuration
DB and SMTP credentials are loaded from the `fw_configs` table.
See the EIT Framework project settings for configuration.
